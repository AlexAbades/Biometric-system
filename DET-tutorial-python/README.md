# DET tutorial for students
An interactive notebook for students to learn about score distributions, decision thresholds, type I and type II errors, and DET curves.

## Dependencies
* Python 3
* jupyter
* matplotlib
* matplotlib2tikz
* numpy
* pandas
* scipy
* seaborn

## Installation
Run `pip install -r requirements.txt`
